## 2. Getting Started

**Step 1: Access the System**
- Open your web browser (Chrome, Firefox, Edge, or Safari)
- Go to: `http://localhost:3000` (if running locally)  
  OR  
  Visit the live version: [https://edumanage](https://edumanage.app)

**Step 2: Navigate the System**
- Use the **navbar** at the top to switch between features
- Use the **sidebar** for quick access to tools
- Click any button or link to perform actions

1. Click **"Dashboard"** in the top menu
2. View:

## 5. Grade Analysis

**What it does:**  
Record student grades and see their performance trends.

**How to record grades:**
1. Click **"Grade Analysis"** in the top menu
2. Select the **student** from the dropdown (e.g., John Doe)
3. Select the **subject** (e.g., Mathematics)
4. Enter the **marks** (0-100)
5. Click **"Save"**

**How to view performance:**
1. Go to **Grade Analysis**
2. Select a student and subject
3. View:
   - Current grade
   - Grade trend over time (graph)
   - Average grade across subjects

**Example:**
- Student: John Doe
- Subject: Mathematics
- Marks: 85
- System shows: "distinction"

## 6. Student Records

**What it does:**  
Add, edit, and manage student information.

**How to add a student:**
1. Click **"Student Records"** in the top menu
2. Click **"Add Student"**
3. Fill in:
   - **Student Name** (e.g., Jane Smith)
   - **grade** (e.g., 10)
   - **subject** (e.g, english)
   - **mark** (e.g, 78)
   - **attendance** (e.g., 90)
   4. Click **"Save"**

**How to edit a student:**
1. Go to **Student Records**
2. Find the student in the list
3. Click **"pencil icon"**
4. Make changes
5. Click **"Save"**

**How to delete a student:**
1. Go to **Student Records**
2. Find the student
3. Click **"bin icon"**
4. Confirm deletion

**How to search for a student:**
- go to records page and find the search section
- Type:
  - Student name (e.g., "John")
  - subject (e.g., mathematics)

## 8. Troubleshooting

### Issue: Page won’t load
**Solution:**
- Refresh the page (press F5)
- Clear browser cache (Ctrl+Shift+Delete)
- Try a different browser

### Issue: Can’t save data
**Solution:**
- Make sure all required fields are filled
- Click the **"Save"** button after making changes
- Try refreshing the page

### Issue: Can’t find a student
**Solution:**
- Use the search bar
- Try searching by name, or subject
- Check spelling

### Issue: Attendance percentage not showing
**Solution:**
- Make sure you saved the attendance
- Wait a few seconds — the system calculates automatically

### Issue: Grade trend not updating
**Solution:**
- Make sure you saved the grade
- Check if you have multiple grades for the same subject
